Module br.player.known

These functions help in retrieving information about spell knowledge.

Known functions are stored in br.player.known and can be utilized by `local known = br.player.known` in your profile. For spell known, `spell` in the function represent the name in the actions list (Spec, Shared Class, Shared Global Lists) defined in System/List/Spells.lua

Functions

known.spell () Checks if spell is known or not.
